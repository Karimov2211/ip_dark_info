# Тарзи установка:
Kali linux /  Parrot OS

1) git clone https://github.com/Karimov2211/ip_dark_info.git
2) cd ip_dark_info

3) pip3 install pyfiglet

4) pip3 install folium

5) pip3 install requests

6) pip3 install colorama

7) python3 probiv.py
